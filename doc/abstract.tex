We provide a library and command-line interface for TA bookings,
schedules, time tracking, task management, GitHub integration,
and agent-assisted task processing.

The library can parse ICS calendars containing the course schedule, generate a
sign-up sheet for TAs, later parse that sign-up sheet (directly from a Google
Sheets URL) to compute employment data for amanuensis or time sheets for hourly
employed TAs, among other things.

The schedule view integrates course events, external calendars, and pending
tasks, automatically scheduling work items into free time slots to give a
complete picture of upcoming commitments.

It also supports delegated task execution by AI agents when automation is
useful.
