We provide a library and command-line interface for managing TA bookings,
course schedules, sign-up sheets, employment contracts, time tracking, and
task management.
%
The CLI follows Git's model of composable subcommands---each concern
(courses, sign-up sheets, HR, scheduling, time tracking, tasks) lives in its
own module and can be used standalone or from a shell script, so that
repetitive workflows can be handed off to Cron or AI agents.
%
The schedule view integrates course events, external ICS calendars, and
pending tasks to give a complete picture of upcoming commitments.
%
The library can parse ICS calendars containing the course schedule, generate a
sign-up sheet for TAs, later parse that sign-up sheet (directly from a Google
Sheets URL) to compute employment data for amanuensis contracts or time sheets
for hourly-employed TAs.
%
Time tracking supports both plain label tracking and launch-based workflows,
where a command or tmux shell is tied to the tracking session.
